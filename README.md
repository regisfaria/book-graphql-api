# Books API

## Overview

This api is developed with NestJS, Typescript, PostgreSQL and GraphQL with the purpose of studying these techs.

## Installation & Running

- You will need:
  - PostgreSQL
  - Nodejs
  - NPM

- install dependencies with `npm install`
